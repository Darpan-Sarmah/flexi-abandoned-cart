# Changelog

All notable changes to Flexi Abandoned Cart Recovery for WooCommerce are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-01

### Added
- Cart abandonment tracking via WooCommerce checkout hooks.
- Automated recovery email sequences with configurable time intervals.
- Analytics dashboard showing abandoned cart counts, recovery rates, and revenue recovered.
- Dynamic coupon generation for use as incentives in recovery emails.
- Email performance tracking including open rates, click-through rates, and conversions.
- Global settings panel for abandonment threshold, email schedule, and coupon configuration.
- Logs view for debugging and auditing email sending activity.
- Translation-ready codebase with `.pot` template file in `languages/`.
- Graceful deactivation if WooCommerce is not active.
