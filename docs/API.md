## Overview

Flexi Abandon Cart Recovery follows the standard WordPress plugin architecture generated by the [WordPress Plugin Boilerplate](https://wppb.me/). All functionality is encapsulated in classes located in the `includes/`, `admin/`, and `public/` directories.

### Directory Structure

```
flexi-abandon-cart-recovery/
├── flexi-abandon-cart-recovery.php   # Bootstrap file, constants, hooks
├── includes/
│   ├── class-flexi-abandon-cart-recovery.php           # Core plugin class
│   ├── class-flexi-abandon-cart-recovery-activator.php # Activation logic
```

## Core Classes

### Flexi_Abandon_Cart_Recovery

**File:** `includes/class-flexi-abandon-cart-recovery.php`

The central plugin class. Instantiated by `run_flexi_abandon_cart_recovery()` and responsible for loading all dependencies and registering all hooks.

#### Properties

| Property | Type | Access | Description |
|---|---|---|---|
| `$loader` | `Flexi_Abandon_Cart_Recovery_Loader` | protected | Hook loader instance |
| `$plugin_name` | `string` | protected | Plugin identifier (`flexi-abandon-cart-recovery`) |
| `$version` | `string` | protected | Plugin version string |

#### Public Methods

| Method | Return | Description |
|---|---|---|
| `__construct()` | void | Loads dependencies, sets locale, defines admin and public hooks |
| `run()` | void | Executes the loader to register all hooks with WordPress |
| `get_plugin_name()` | string | Returns `$plugin_name` |
| `get_version()` | string | Returns `$version` |
| `get_loader()` | `Flexi_Abandon_Cart_Recovery_Loader` | Returns the loader instance |

---

### Flexi_Abandon_Cart_Recovery_Admin

**File:** `admin/class-flexi-abandon-cart-recovery-admin.php`

Handles all admin-area functionality: menu registration, settings saving, cart tracking, email dispatch, and cron callbacks.

#### Key Methods

| Method | Description |
|---|---|
| `enqueue_styles()` | Enqueues admin CSS |
| `enqueue_scripts()` | Enqueues admin JavaScript |
| `flexi_acr_admin_menu()` | Registers top-level and sub-page admin menu items |
| `flexi_capture_abandoned_cart_data()` | Captures cart data when WooCommerce cart events fire |
| `flexi_check_for_abandoned_carts()` | Cron callback — checks for abandoned carts and dispatches emails |
| `flexi_save_trigger_data()` | AJAX handler — saves email trigger configuration |
| `flexi_woocommerce_get_all_products()` | AJAX handler — returns all WooCommerce products as JSON |
| `flexi_woocommerce_get_all_categories()` | AJAX handler — returns all WooCommerce categories as JSON |
| `flexi_woocommerce_get_all_user_roles()` | AJAX handler — returns all WordPress user roles as JSON |
| `flexi_send_test_mail()` | AJAX handler — sends a test email using a given template |
| `flexi_send_report_over_mail()` | AJAX handler — emails an analytics report |
| `track_email_open()` | Admin-post callback — records email open event |
| `track_link_click()` | Hook callback — records link click and marks cart as accessed |
| `track_purchase()` | `woocommerce_thankyou` callback — marks cart as purchased |
| `flexi_add_custom_scheduler()` | `cron_schedules` filter callback — adds custom cron intervals |
| `flexi_coupon_cart_expiry_scheduler()` | `init` callback — registers cron events if not already scheduled |
| `mark_flexi_cart_expiry()` | Cron callback — marks expired carts |
| `mark_flexi_coupons_expiry()` | Cron callback — marks expired coupons |
| `flexi_tinymce_admin_btn()` | `mce_buttons` filter — adds custom TinyMCE buttons |
| `flexi_admin_filter_mce_plugin()` | `mce_external_plugins` filter — loads custom TinyMCE plugins |

## Database Schema

### `{prefix}flexi_users_cart_details`

| Column | Type | Description |
|---|---|---|
| `id` | BIGINT UNSIGNED AUTO_INCREMENT | Primary key |
| `user_id` | BIGINT UNSIGNED | WordPress user ID (0 for guest) |
| `cart_status` | VARCHAR(20) | `active`, `abandoned`, `recovered`, `purchased` |
| `created_at` | DATETIME | Cart creation timestamp |
| `abandon_time` | DATETIME | Timestamp when cart was marked abandoned |
| `last_interaction_at` | DATETIME | Last cart interaction timestamp |
| `is_expired` | TINYINT(1) | 1 if cart has expired |
| `is_hidden` | TINYINT(1) | 1 if cart is hidden from dashboard |
| `is_purchased` | TINYINT(1) | 1 if associated order was completed |
| `extra_data` | TEXT | JSON-encoded additional data |

### `{prefix}flexi_abandon_cart_users`

| Column | Type | Description |
|---|---|---|
| `id` | BIGINT UNSIGNED | WordPress user ID (primary key) |
| `user_information` | LONGTEXT | JSON-encoded user data (email, name, roles) |

### `{prefix}flexi_users_cart_items`

| Column | Type | Description |
|---|---|---|
| `id` | BIGINT UNSIGNED AUTO_INCREMENT | Primary key |
| `cart_id` | BIGINT UNSIGNED | Foreign key → `flexi_users_cart_details.id` |

| `quantity` | INT | Item quantity |

### `{prefix}flexi_email_logs`

| Column | Type | Description |

## AJAX Endpoints

All AJAX endpoints are registered in `Flexi_Abandon_Cart_Recovery::define_admin_hooks()`.

| Action | Handler Method | Access |
|---|---|---|
| `flexi_save_trigger_data` | `flexi_save_trigger_data` | Admin only |
| `woocommerce_get_all_products` | `flexi_woocommerce_get_all_products` | Admin + nopriv |
| `woocommerce_get_all_categories` | `flexi_woocommerce_get_all_categories` | Admin + nopriv |
| `woocommerce_get_all_roles` | `flexi_woocommerce_get_all_user_roles` | Admin + nopriv |
| `flexi_send_test_mail` | `flexi_send_test_mail` | Admin + nopriv |
| `flexi_send_report_over_mail` | `flexi_send_report_over_mail` | Admin + nopriv |

All endpoints require a valid WordPress nonce to prevent CSRF.

---

## Admin-Post Endpoints

| Action | Handler Method | Description |
|---|---|---|
| `track_email_open` | `track_email_open` | Records email open event via tracking pixel |

The open-tracking pixel URL is embedded in all outgoing emails. The endpoint is accessible to both logged-in and anonymous users (`admin_post` + `admin_post_nopriv`).
