# Release Notes

This file contains release notes for each version of Flexi Abandoned Cart Recovery for WooCommerce.

---

## [1.0.0] – 2024-01-01

### Features Added
- Cart abandonment tracking via WooCommerce checkout hooks.
- Automated recovery email sequences with configurable time intervals.
- Analytics dashboard showing abandoned cart counts, recovery rates, and revenue recovered.
- Dynamic coupon generation for use as incentives in recovery emails.
- Email performance tracking: open rates, click-through rates, and conversion counts.
- Global settings panel covering abandonment threshold, email schedule, and coupon rules.
- Logs view for auditing email sending activity.

### Bugs Fixed
- Initial release; no previous bugs to fix.

### Improvements
- Translation-ready codebase with `.pot` template included in `languages/`.
- Graceful deactivation notice if WooCommerce is not installed or active.

### Breaking Changes
- None (initial release).

### Upgrade Instructions
- Not applicable for initial release.

### Known Issues
- None at time of release.
