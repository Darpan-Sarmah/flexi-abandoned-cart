# Support

Thank you for using Flexi Abandoned Cart Recovery for WooCommerce.

## Getting Help

### Documentation

Before raising a support request, please check the documentation included with the plugin:

- [`docs/FAQ.md`](FAQ.md) – Answers to common questions.
- [`docs/TROUBLESHOOTING.md`](TROUBLESHOOTING.md) – Step-by-step guides for common issues.
- [`docs/UPGRADE-GUIDE.md`](UPGRADE-GUIDE.md) – Instructions for upgrading between versions.

### WordPress.org Support Forum

For general usage questions, the WordPress.org support forum is the best place to ask:

👉 https://wordpress.org/support/plugin/flexi-abandon-cart-recovery/

Please search existing topics before opening a new one.

### GitHub Issues

For confirmed bugs or feature requests, please use the GitHub issue tracker:

👉 https://github.com/Darpan-Sarmah/flexi-abandoned-cart/issues

When reporting a bug, please include:
- Plugin version
- WordPress version
- WooCommerce version
- PHP version
- Steps to reproduce the issue
- Any relevant error messages or screenshots

## Before Requesting Support

Please try the following first:

1. **Deactivate other plugins** temporarily to check for conflicts.
2. **Switch to a default theme** (e.g. Twenty Twenty-Four) to rule out theme conflicts.
3. **Enable WP_DEBUG** in `wp-config.php` and check `wp-content/debug.log` for PHP errors.
4. **Clear your caches** (browser cache, WordPress object cache, page cache).
5. **Check the Flexi Logs** page in the WordPress admin for any error messages from the plugin.

## Response Times

Support requests are typically responded to within 2–3 business days. Response times may vary.
